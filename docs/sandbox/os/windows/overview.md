# Sandbox Windows

1. Windows Sandbox
- Der Grund, warum Windows Sandbox besser ist als Sandboxie, ist, dass sie auf einer höheren Ebene läuft.

Sandboxie arbeitet intern auf der User-Ebene, während Windows Sandbox ein eigener Hyper-Visor-Bereich ist und dementsprechend noch sicherer ist.
